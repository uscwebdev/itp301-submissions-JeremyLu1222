Project Summary

Home page

This project is about booking private airline for users. At the main page, user can see two clickable link marked by underline. This will lead user to two seperate pages which are booking page and about us page.

Book Page

The book page contains four buttons not for clicking, but when user hover mouse on those button, the destination graph will change. And also there is a clickable book button, allowing user to put their information inside. And it will validate user input to make sure correct information. As always, nav link will direct user to other pages.

About page

About page contains two part. First is a fleet gallery that user can hover their mouse onto the graph to show the brand of the plane. the second is a service gallery each contains a clickable link. When user clicks it, the detail description will pop up in the middle of the screen.

Extra used

Only React Router library (counts as an extra).
